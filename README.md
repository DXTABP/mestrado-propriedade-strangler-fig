# Proposta de Pesquisa de Mestrado - Arquiteto de Software

Este repositório contém a proposta de pesquisa individual focada em estratégias de modernização de sistemas.

# Projeto
Título: Avaliação do Padrão Strangler Fig na Modernização Incremental de Sistemas Monolíticos.

# Resumo Técnico
Pesquisa voltada para a decomposição de monólitos legados utilizando o padrão **Strangler Fig** e **Domain-Driven Design (DDD)**. O objetivo é criar um framework que permita a migração para microsserviços sem interromper a operação de negócio.
